# 음식과 허기

<img src="../assets/player-guides/openmoji/food.svg" alt="빵 아이콘" width="96">

이 문서는 **Minecraft Java Edition 26.1 / Paper** 기준입니다. `허기` 2점은 화면의
닭다리 아이콘 1개입니다. `포화도`는 보이지 않는 예비 게이지로, 포화도가 남아 있는
동안에는 허기가 바로 줄지 않습니다.

## 허기·포화도 작동 방식

- 허기와 포화도의 최대치는 각각 20입니다. 포화도는 현재 허기보다 높아질 수 없습니다.
- 달리기, 점프, 수영, 채굴, 공격, 피해 후 자연 회복 등이 피로도를 쌓습니다. 피로도가
  누적되면 포화도부터 줄고 그다음 허기가 줄어듭니다.
- 허기가 18 이상이면 `naturalRegeneration=true`일 때 자연 회복할 수 있습니다.
  포화도가 높을수록 처음 회복이 빠릅니다.
- 허기가 6 이하이면 달리기를 시작할 수 없습니다. 허기 0에서는 아사 피해를 받으며,
  Java의 쉬움은 체력 10, 보통은 체력 1에서 멈추고 어려움은 사망할 수 있습니다.
- 배가 거의 찼을 때는 허기 회복량보다 포화도가 높은 음식을 먹는 편이 효율적입니다.

## 일반 음식 수치

![주요 음식의 포화도를 비교한 한영 막대 차트](../assets/player-guides/diagrams/food-efficiency.svg)

통째로 먹을 때의 음식 수치는 해당 생선과 같지만 양동이째 음식으로 쓸 이유는 거의

14, 포화도 2.8이지만 들고 다니며 먹을 수 없고 회수도 못 합니다.

## 위험한 음식

| 음식 | 수치 | 추가 효과 |
|---|---|---|
| 썩은 살 | 허기 4 / 포화 0.8 | 80% 확률로 허기 30초. 비상식 외에는 성직자 판매 추천 |
| 독이 있는 감자 | 허기 2 / 포화 1.2 | 60% 확률로 독 5초 |
| 거미 눈 | 허기 2 / 포화 3.2 | 항상 독 5초; 양조 재료로 보관 추천 |
| 복어 | 허기 1 / 포화 0.2 | 독 II 60초, 허기 III 15초, 멀미 15초 |
| 생닭고기 | 허기 2 / 포화 1.2 | 30% 확률로 허기 30초 |

독은 일반적으로 체력을 1 아래로 내리지 않지만 다른 피해와 겹치면 위험합니다. 꿀이 든
병은 독만 지우고, 우유는 다른 상태 효과까지 모두 지웁니다.

## 수상한 스튜 꽃 효과

스튜 자체 이름만으로는 효과를 구분할 수 없습니다. 직접 제작할 때 사용한 꽃을
기억하거나 상자를 나눠 보관하세요. 시간은 Java 26.1 기준입니다.

| 꽃 | 효과 |
|---|---|
| 민들레·황금 민들레·파란색 난초 | 포화 0.35초(즉시 허기·포화 회복) |
| 알리움 | 화염 저항 3초 |
| 선애기별꽃·뜬 눈망울꽃 | 실명 11초 |
| 감은 눈망울꽃 | 멀미 7초 |
| 수레국화 | 점프 강화 5초 |
| 은방울꽃 | 독 11초 |
| 빨강·주황·분홍·하양 튤립 | 나약함 7초 |
| 데이지 | 재생 7초 |
| 양귀비·횃불꽃 | 야간 투시 5초 |
| 위더 장미 | 시듦 7초 |

갈색 무시룸에 꽃을 먹인 뒤 그릇을 사용해도 해당 꽃 효과의 수상한 스튜를 한 번 얻을
수 있습니다.

## 상황별 추천

- 평상시·장거리: 황금 당근, 스테이크, 익힌 돼지고기.
- 자동 농장 중심: 빵 또는 구운 감자로 시작해 농부 달인에게 황금 당근 구매.
- 전투: 일반 음식과 별도 슬롯에 황금 사과. 황금 사과는 허기보다 효과를 위해 사용.
- 비상식: 썩은 살은 먹은 뒤 허기 효과 중 불필요한 달리기·점프를 줄입니다.
- 네더·엔드: 포화도 높은 중첩 음식과 상황별 [포션](brewing.md)을 함께 준비합니다.
