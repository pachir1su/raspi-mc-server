# 주민 거래 안내

<img src="../assets/player-guides/openmoji/villager-trading.svg" alt="악수 아이콘" width="96">

이 문서는 **Minecraft Java Edition 26.1 / Paper**의 기본 주민 거래를 설명합니다.
26.1부터 거래 정의가 데이터 기반으로 바뀌었지만, 별도 데이터 팩을 설치하지 않은 이
서버에서는 바닐라 26.1 거래 표를 사용합니다. 과거의 실험적 `주민 거래 재조정` 표와
혼동하지 마세요.

## 직업 지정과 거래 고정

1. 직업 없는 성인 주민이 작업소 블록에 접근하면 직업을 얻습니다. 초록 옷의
   멍청이(Nitwit)는 직업과 거래를 가질 수 없습니다.
2. **한 번이라도 거래하면** 그 주민의 직업과 이미 생성된 거래가 영구 고정됩니다.
3. 첫 거래 전에는 작업소를 부쉈다가 다시 놓아 거래를 다시 뽑을 수 있습니다. 원하는
   거래를 확인하기 전에 실수로 거래하지 마세요.
4. 거래를 모두 쓰면 잠기며, 주민이 자기 작업소에 접근해 일할 때 하루 최대 두 번
   재입고할 수 있습니다. 침대보다 **연결된 작업소에 실제로 닿는 경로**가 중요합니다.

주민 등급은 초보자→수습생→기능공→전문가→달인 순서입니다. 거래로 경험치를 채우면
다음 등급 거래가 열리며, 열린 거래는 주민이 죽지 않는 한 사라지지 않습니다.

## 직업별 핵심 거래

가격과 실제 제안 조합은 주민마다 달라지고 수요·평판에 따라 변하므로, 아래는 고정
가격표가 아니라 26.1 기본 거래 풀에서 특히 쓸 만한 항목입니다.

|---|---|---|---|
| 갑옷 제조인 (Armorer) | 용광로 | 석탄·철·용암 양동이·다이아몬드 | 철/사슬 방어구, 방패, 인챈트된 다이아몬드 방어구 |
| 도살업자 (Butcher) | 훈연기 | 생고기·석탄·말린 켈프 블록·달콤한 열매 | 익힌 고기, 토끼 스튜 |
| 지도 제작자 (Cartographer) | 지도 제작대 | 종이·유리판·나침반 | 빈 지도, 여러 마을/늪/정글/해저 유적/시련의 회당/삼림 대저택 지도, 현수막 |

26.1의 기본 사서는 초보자~전문가 등급에서 무작위 인챈트 책 거래를 제안할 수
있습니다. `수선`을 포함한 거래 가능한 보물 인챈트도 나올 수 있지만, 영혼 가속·신속한
잠행·돌풍은 사서 거래 대상이 아닙니다. 자세한 목록은 [인챈트](enchantments.md)를
참고하세요.

## 가격이 바뀌는 이유

- **수요:** 같은 거래를 많이 소진하면 재입고 후 가격이 오를 수 있습니다. 며칠 쉬거나
  다른 거래를 사용하면 수요가 완화될 수 있습니다.
- **평판:** 주민을 때리거나 죽이면 주변 주민의 가격이 나빠질 수 있습니다. 거래와
  좀비 주민 치료는 평판을 좋게 만듭니다.
- **마을의 영웅:** 습격에 승리하면 효과가 지속되는 동안 할인을 받습니다.
- **치료 할인:** 좀비 주민을 치료한 첫 보상은 영구 할인입니다. Java 1.20.2부터 같은
  주민을 반복 감염·치료해 할인을 계속 중첩하는 방식은 작동하지 않습니다.

표시 가격은 최소 1개보다 내려가지 않으며, 수요가 높은 거래는 기본 가격보다 비싸질
수 있습니다.

## 좀비 주민 치료

1. 좀비 주민을 햇빛과 다른 몹에게서 보호해 가둡니다.
2. **투척용 나약함의 물약**을 맞힙니다.
3. 나약함 상태인 좀비 주민에게 **일반 황금 사과**를 사용합니다.
4. 흔들림과 붉은 입자가 끝날 때까지 약 3~5분 안전하게 기다립니다. 철창과 침대가
   주변에 있으면 조금 빨라질 수 있습니다.

난이도별로 주민이 좀비에게 죽었을 때 좀비 주민이 될 확률은 쉬움 0%, 보통 50%,
어려움 100%입니다. 할인 때문에 일부러 감염시킬 때 쉬움·보통에서는 주민을 완전히
잃을 수 있습니다. 이 서버의 난이도를 먼저 확인하세요.

## 거래소를 안전하게 만들기

- 주민 한 명당 작업소 하나를 두고, 블록 앞에 서서 초록 작업 입자가 나는지 확인합니다.
- 작업소를 옮길 때는 어떤 주민이 연결됐는지 하나씩 확인합니다. 가까운 블록이라고
  반드시 눈앞의 주민이 연결되는 것은 아닙니다.
- 번개는 주민을 마녀로 바꿀 수 있으므로 지붕과 피뢰침을 설치합니다.
- 좀비 치료 공간은 거래 주민과 분리하고 난이도·탈출·햇빛 차단을 먼저 점검합니다.
- 많은 주민과 호퍼·레드스톤을 한 청크에 몰면 Raspberry Pi의 틱 비용이 커집니다.
  필요한 직업과 거래만 유지하고 이동 경로를 단순하게 만드세요.

## 추천 우선순위

1. 화살 제조인에게 막대기를 팔아 초기 에메랄드를 마련합니다.
2. 농부에게 자동 농장 작물을 팔고 황금 당근을 안정적으로 구합니다.
3. 사서에서 수선·내구성·효율·보호 계열 책을 확보합니다.
4. 갑옷/도구/무기 대장장이를 달인까지 올려 다이아몬드 장비를 재생 가능 자원으로
   만듭니다.
5. 성직자에게 썩은 살을 팔고 엔더 진주와 경험치 병을 구합니다.
