## Minecraft status-effect icons

The 40 PNG files in `player-guides/status-effects/` are Minecraft Java Edition
26.1 status-effect UI icons supplied by a repository contributor after being
collected from the [NamuWiki status-effect page](https://namu.wiki/w/%EB%A7%88%EC%9D%B8%ED%81%AC%EB%9E%98%ED%94%84%ED%8A%B8/%EC%83%81%ED%83%9C%20%ED%9A%A8%EA%B3%BC).
Their filenames were normalized for this repository; the image contents were
not modified. Each supplied image was matched against the corresponding icon in
the official Minecraft Java 26.1 client assets before use.

## Minecraft brewing images

The 85 PNG files in `player-guides/brewing/` are drinkable, splash, and
lingering potion images plus a brewing-stand interface and core ingredient
images supplied by a repository contributor after being collected from the
[NamuWiki food and drink page](https://namu.wiki/w/%EB%A7%88%EC%9D%B8%ED%81%AC%EB%9E%98%ED%94%84%ED%8A%B8/%EC%95%84%EC%9D%B4%ED%85%9C/%EC%9D%8C%EC%8B%9D%20%EB%B0%8F%20%EC%9D%8C%EB%A3%8C#s-10.1)
and related Minecraft item entries.
Their filenames were normalized for this repository; the image contents were
not modified. The supplied order and colors were checked against the potion and
effect registries in the official Minecraft Java 26.1 client. The Bedrock-only
Potion of Decay images were intentionally excluded from this Java/Paper guide.
